The Client

Downloading and Installing

You can find the latest client downloads at our client development site. Since the code is written in the Python Programming Language it should be usable anywhere that Python is installed.


To install the client you require the following Python libraries to be available: 


- setuptools 0.6c9+
- sqlalchemy 5.8+
If possible, it's best to install these libraries through your Operating Systems package manager. Next you must extract the files from the archive. Depending on what OS you use the operation will be different. Then you must run ./setup.py install as a root/Administrator user. The last thing you must do is copy the conf/example.cfg to the location you would like to keep the config file and edit it to your liking.


Using

Upon first use you should update the local database. To do this you can run victims-update-db and pass it the location of your configuration file. Here is an example on my Fedora laptop: 

[steve@oa .~]$ victims-update-db -c ~/example.cfg 
[steve@oa .~]$

In normal Unix fashion, no output means everything went well! The command you have just done should be executed from time to time to make sure you have all the latest hashes to check against. Now it's time to scan with victims-scan!
