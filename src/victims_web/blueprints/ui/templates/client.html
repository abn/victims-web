Client

Downloading and Installing

You can find the latest client downloads at our client development site. Since the code is written in python, it should be usable anywhere that python is installed.


To install the client you require the following Python libraries to be available: 


- setuptools 0.6c9+
- sqlalchemy 5.8+
If possible, install these libraries using your operating system's package manager. Next, extract the files from the client archive and run the setup.py script as the root/Administrator use. Finally, copy conf/example.cfg to the preferred location and edit it to apply your settings.


Using

Upon first use you should update the local database. To do this you can run victims-update-db and pass it the location of your configuration file. For example: 
$ victims-update-db -c ~/example.cfg

If there is no output, the command completed successfully. This command should be repeated periodically to keep your local database up to date with the master database. Now you're ready to perform a scan using victims-scan: 

$ victims-scan -l -c ~/example.cfg /tmp/
apache-tomcat-5.5.12/server/lib/catalina.jar (inside /tmp/apache-tomcat-5.5.12.zip):
    CVE-2009-2693,CVE-2009-2901,CVE-2009-2902
/tmp/apache-tomcat-5.5.12/server/lib/catalina.jar:
    CVE-2009-2693,CVE-2009-2901,CVE-2009-2902
Scanned 92 packages

For more information on the available switches for each command, run the command with no options (or use --help).
